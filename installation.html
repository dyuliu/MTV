Installation¶

Stable release¶

To install mtv, run this command in your terminal:

$ pip install mtv

This is the preferred method to install mtv, as it will always install the most recent stable release.

If you don’t have pip installed, this Python installation guide can guide you through the process.

From source¶

The source for mtv can be downloaded from the Github repo.

You can either clone the public repository:

$ git clone git://github.com/HDI-Project/MTV

Or download the tarball:

$ curl  -OL https://github.com/HDI-Project/MTV/tarball/master

Once you have a copy of the source, you can install it with:

$ make install
